# [WIP] Open Film - Open Source Video Editor

Open Film is open source video editor written in Rust.

Currently, only technical verification of video playback is implemented.
Powered by Rust, Iced, rust-ffmpeg.

The video taken from here: https://pixabay.com/ja/videos/%E8%AB%B8%E5%B3%B6-%E6%B5%B7-%E6%97%A5%E3%81%AE%E5%87%BA-%E9%A2%A8%E6%99%AF-%E6%9E%97-2119/

# 日本語での説明

Open FilmはRustで書かれたオープンソースの動画編集ソフトです。

現在、動画再生の技術検証のみ実装されています。
利用しているライブラリはIced、rust-ffmpegです。
